# Synthetic asset provenance

Every geography, contour, route, terminal, zone, duration, noise boundary, energy value, and disruption shown in this project is wholly synthetic and created for this fictional showcase.

The region geometry is defined procedurally in `src/data/region.js`. SVG contours and Three.js elevation are generated from the same normalized local coordinate system and deterministic height functions. Corridor polylines, terminal markers, sensitive zones, predefined lens stops, evidence values, table rows, and exported summaries are also driven by that single source.

No real map tiles, municipality boundaries, resorts, transport routes, operators, aircraft, approvals, forecasts, environmental studies, or client data are used. The planning-table scene is assembled from local SVG and procedural WebGL geometry. It is concept material, not documentary imagery.
